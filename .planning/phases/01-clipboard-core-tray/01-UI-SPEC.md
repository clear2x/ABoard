## Color

Dark theme (Phase 1 is dark-only; light theme deferred to Phase 7).

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #111827 (gray-900) | Window background, main surface |
| Secondary (30%) | rgba(31,41,55,0.5) (gray-800/50) | Card backgrounds |
| Accent (10%) | #3b82f6 (blue-500) | Type badge for "text" items, primary interactive highlight |
| Destructive | #ef4444 (red-500) | Not used in Phase 1 — reserved |

Accent reserved for:
- "text" type badge background (blue-500/20 text-blue-400)
- Not used for buttons or CTAs in Phase 1 (no interactive accent elements yet)

Semantic type badge colors:
- Text: blue-500/20 background, blue-400 text
- Image: green-500/20 background, green-400 text
- File paths: purple-500/20 background, purple-400 text

Border tones:
- Card borders: gray-700/50 (default), gray-600/50 (hover)
- Header border: gray-800
- Dividers: gray-800

Text hierarchy:
- Primary: white (header title)
- Secondary: gray-300 (card content preview)
- Tertiary: gray-400 (subtitle, secondary info)
- Muted: gray-500 (timestamps, empty state)

## Layout Contract

### Window Dimensions
- Default: 400px wide x 500px tall
- Resizable: true
- Position: centered on cursor (per CONTEXT.md)
- Decorations: native window chrome (title bar close/minimize buttons)

### Component Layout

```
+------------------------------------------+
| Header (px-4 py-3, border-b)             |
| [ABoard]              [Clipboard Manager] |
+------------------------------------------+
| ClipboardList (flex-1, overflow-y-auto)  |
|  +--------------------------------------+|
|  | ClipboardItemCard (p-3, rounded-lg)  ||
|  | [text badge]          [12:34:56 PM]  ||
|  | Preview text content...              ||
|  +--------------------------------------+|
|  +--------------------------------------+|
|  | ClipboardItemCard                    ||
|  | ...                                  ||
|  +--------------------------------------+|
|                                          |
| Empty state (centered, gray-500):        |
| "No clipboard items yet. Copy something!"|
+------------------------------------------+
```

### ClipboardItemCard Structure
- Outer: `p-3 bg-gray-800/50 rounded-lg border border-gray-700/50 hover:border-gray-600/50 transition-colors`
- Top row: flex, justify-between
  - Left: type badge (`text-xs px-2 py-0.5 rounded-full` + semantic color)
  - Right: timestamp (`text-xs text-gray-500`, formatted as HH:MM:SS)
- Content: `text-sm text-gray-300 break-all leading-relaxed`
  - Text: truncated to 100 chars + "..."
  - Image: displays "[Image]"
  - File paths: truncated to 100 chars + "..."

### ClipboardList Structure
- Outer: `flex flex-col gap-2 overflow-y-auto max-h-[450px] p-2`
- Uses Solid.js `<Show>` for empty state, `<For>` for item rendering
- Items render newest-first (array order from store)

## Interaction States

| Element | States | Visual Feedback |
|---------|--------|-----------------|
| ClipboardItemCard | default, hover | Hover: border transitions to gray-600/50 |
| Type badge | static | No interactive states — purely informational |
| Empty state | static | No interactive states |
| Window close | click | Hides to tray (not visible in UI) |
| Tray icon | click | Shows context menu |
| Global shortcut | Cmd+Shift+V | Toggles window visibility (no visual feedback in window) |

No loading states in Phase 1 — clipboard data arrives via real-time events.
No selection or click-to-action states — deferred to Phase 3.
