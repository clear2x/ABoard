2. **创建格式化工具 store** (src/stores/format-tools.ts):

   所有工具为纯 JS 实现，不调用后端。

   a. **JSON 工具**:
   ```typescript
   export function formatJson(content: string): { result: string } | { error: string; line?: number; column?: number }
   ```
   - 美化: `JSON.stringify(JSON.parse(content), null, 2)`
   - 捕获 SyntaxError，解析错误位置 (从 error.message 中提取行号列号)
   - 返回 `{ result: formatted }` 或 `{ error: "...", line: N, column: N }`

   ```typescript
   export function minifyJson(content: string): { result: string } | { error: string }
   ```
   - 压缩: `JSON.stringify(JSON.parse(content))`

   ```typescript
   export function validateJson(content: string): { valid: true } | { valid: false; error: string; line?: number; column?: number }
   ```
   - 仅校验不格式化
   - 解析 JSON 解析错误信息中的行号/列号

   b. **XML 工具**:
   ```typescript
   export function formatXml(content: string): { result: string } | { error: string; line?: number }
   ```
   - 使用 DOMParser 解析 XML
   - 递归遍历节点树，生成缩进格式输出
   - 校验: DOMParser 解析失败时返回 `<parsererror>` 节点，提取错误信息

   ```typescript
   export function validateXml(content: string): { valid: true } | { valid: false; error: string; line?: number }
   ```

   c. **格式转换**:
   ```typescript
   export type FormatType = "markdown" | "html" | "plaintext"

   export function convertFormat(
     content: string,
     from: FormatType,
     to: FormatType
   ): string
   ```
   - markdown -> html: `markdownit().render(content)`
   - html -> markdown: `new TurndownService().turndown(content)`
   - markdown -> plaintext: 先转 html 再 strip tags (`content.replace(/<[^>]*>/g, "")`)
   - html -> plaintext: strip tags
   - plaintext -> html: 包裹 `<p>` 段落标签
   - plaintext -> markdown: 同上（plaintext 无特殊 Markdown 语法）

   d. **内容格式检测辅助函数**:
   ```typescript
   export function detectContentFormat(content: string): { isJson: boolean; isXml: boolean; isMarkdown: boolean; isHtml: boolean }
   ```
   - isJson: 尝试 JSON.parse，成功且非原始值
   - isXml: 以 `<?xml` 或 `<` 开头且包含闭合标签
   - isMarkdown: 检测 Markdown 特征 (#, *, -, `, ``` 等)
   - isHtml: 包含 `<` + `>` 标签对

   e. **行内错误高亮**:
   ```typescript
   export function highlightJsonError(content: string, line?: number, error?: string): string
   ```
   - 在指定行添加错误标记注释 `// ERROR: {message}`
   - 用于 AiResultPopup 中的代码展示
